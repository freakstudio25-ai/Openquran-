# OpenQuran

<p align="center">
  <b>A modern, simple and beautiful Quran app for Android</b>
</p>

## About

OpenQuran is a lightweight Quran application designed to make reading and exploring the Holy Quran simple and accessible.

## Features

- 📖 Read Quran chapters easily
- 🔍 Search verses and chapters
- 🌙 Clean dark mode interface
- 🎧 Quran audio support (coming soon)
- ⭐ Bookmark important verses
- ⚡ Fast and lightweight

## Built With

- Flutter
- Dart

## Installation

1. Download the latest APK from Releases
2. Install it on your Android device
3. Open OpenQuran and start reading
